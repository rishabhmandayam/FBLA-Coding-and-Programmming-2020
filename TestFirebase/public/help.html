How to add a Student
  • Navigate to the Add Student page using the side menu.
  • Enter the student's name, ID, and log any hours they have worked or awards they have earned.
  • Click submit to add the student.
How to add hours to a Student
  • Navigate to the Add Entry page using the side menu.
  • Enter the student's ID and log any hours they have worked or awards they have earned.
  • Log the date of the Entry
  • Click submit to add the Entry
How to add an award to a Student
  • Navigate to the Add Entry page using the side menu.
  • Enter the student's ID and log any hours they have worked or awards they have earned.
  • Log the date of the Entry
  • Click submit to add the Entry
How to view a students total hours
  • Navigate to the Dashboard page using the side menu.
  • Scroll down the table till you find the student you are searching for
  • The total number of hours will be located under the hours column
How to modify data for a Student
  • Navigate to the Dashboard page using the side menu.
  • Click the OFF toggle to turn edit mode ON
  • Edit the data for the student in the table and click Save
How to delete a Student
  • Navigate to the Delete Student page using the side menu.
  • Enter the student ID of the student that will be deleted
  • Click the Delete Student Button
How to export to a PDF
  • Navigate to either the Dashboard, Entries, or Report Pages
  • Click the Export dropdown
  • Select PDF from the dropdown list
How to export to a Word document
  • Navigate to either the Dashboard, Entries, or Report Pages
  • Click the Export dropdown
  • Select Word from the dropdown list
How to export to an Excel file
  • Navigate to either the Dashboard, Entries, or Report Pages
  • Click the Export dropdown
  • Select Excel from the dropdown list
How to print a report
  • Navigate to either the Dashboard, Entries, or Report Pages
  • Click the Print button
  • Select a printer and click print
How to view all entries
  • Navigate to Entries page using the side menu
  • All of the entries will be under the Student Information table
How to view all entries for a month
  • Navigate to Monthly Reports Page
  • Select the month you would like to view
  • Click the Generate Report button
How to view all entries for an year
  • Navigate to Annual Reports Page
  • Select the year you would like to view
  • Click the Generate Report button
How is my data secured?
  • All of the data is secured by Firebase
  • To access the data, a user has to have a set of unique config files
  • Even if they have these config files, they need to be authenticated by Google
